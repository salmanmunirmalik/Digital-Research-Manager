## ğŸš€ Getting Started

### **Prerequisites**
- Node.js 18+ and npm
- PostgreSQL database
- Modern web browser (Chrome, Firefox, Safari, Edge)

### **Installation & Setup**
1. Clone the repository
2. Install dependencies with `npm install`
3. Configure database connection
4. Start development server with `npm run dev:frontend`
5. Access application at `http://localhost:5173`

### **Key Navigation Paths**
- **Lab Management**: `/lab-management` - Overview and quick actions
- **Data & Results**: `/data-results` - Data import and management
- **Protocols**: `/protocols` - Protocol library and management
- **Tools & Resources**: `/tools-resources` - Calculator hub and specialized tools
- **Data Analytics**: `/data-analytics` - Advanced statistical analysis
